# Employee Management Test Application

A clean, functional web application built for testing tracking tools like Pendo. Features multiple pages, forms, buttons, tabs, and interactive elements.

## Features

- **Login Screen** - Email/password form with checkbox
- **Dashboard** - Home page with stats cards
- **Employee Directory** - Searchable table with filters and action buttons
- **Employee Form** - Add/edit forms with various input types:
  - Text inputs
  - Email inputs
  - Select dropdowns
  - Radio buttons
  - Checkboxes
  - Date picker
  - Number input
  - Textarea
- **Settings Page** - Tabbed interface with:
  - Profile settings
  - Notification preferences
  - Security settings

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

## Login

Enter any email and password to login (no validation for testing purposes).

## Tech Stack

- React 18
- React Router for navigation
- Vite for build tooling
- Custom CSS (no heavy frameworks)

## Design Philosophy

Clean, professional design with:
- DM Sans typography
- Warm neutral color palette with teal accents
- Consistent spacing and layouts
- Minimal animations
- Focus on functionality over flashiness

Perfect for testing click tracking, form interactions, navigation flows, and user behavior analytics.
